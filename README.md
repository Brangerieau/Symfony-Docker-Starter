# Symfony Docker Starter

⚙️ Installation
--------------

```bash
git clone https://github.com/Brangerieau/Symfony-Docker-Starter.git
```
```bash
docker-compose up -d
# First launch
docker exec -it php_docker_symfony bash
./install.sh
```
- Delete the install.sh file 
- Delete the .git folder
